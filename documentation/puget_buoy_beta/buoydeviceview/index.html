Structure

BuoyDeviceView

Creates the clickable displayed area of a Buoy where you can tap it to connect or move to the screens to deploy the Buoy.

Declaration

struct BuoyDeviceView

Parameters

name

Given name of a Buoy to be displayed.

batteryPercent

Given battery percentage of a Buoy to be displayed.

isConnected

Given parameter of a Buoy to see if it is connected.

showProgressWheel

Boolean variable to assist with showing the progress wheel.

buoy

Given Buoy to observe attributes of for proper display of information.

Overview

Note

This differs from DeviceView because if there are changes made to the underlying Buoy, the changes are reflected in this view, while they are not in DeviceView.
