Structure

StringLabel

This is the view for the string label. It reacts to a user’s touch to change color and show the list of buoys

Declaration

struct StringLabel

Parameters

map

A reference to the mapbox map

isSelected

A Bool value if it is selected

isHighlighted

Similar to isSelected, but this is used by the UI to change colors

buoyString

A reference to the buoy string the label represents

user

An observed instance of the user state

dateFormatter

A DateFormatter used to show the string’s date

searchText

The current search text

proxy

A ScrollViewProxy used to scroll the view and ensure the string label is visible
