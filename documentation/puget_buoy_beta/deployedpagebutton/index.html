DeployedPageButton

A custom button view for the deployed buoy page.

Declaration

struct DeployedPageButton

Parameters

buttonLabel

A string containing the button’s label text.

buttonImage

An image to display on the button.

isSelected

A boolean indicating whether the button is selected.

Overview

The button displays a label and an image, and can be selected or unselected. When selected, the button’s background and text colors are reversed and a blue rectangle is added as an overlay.
