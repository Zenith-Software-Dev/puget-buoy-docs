CountdownClockView

The CountdownClockView struct is a view that displays a countdown clock and allows the user to select a string from a dropdown menu. The struct has several @State properties that store the current time, the seconds, minutes, hours, and days, as well as whether the clock is currently locked. The view also has a switchString() function that allows the user to switch the currently selected string. The view is a NavigationStack and contains several Picker views and buttons for interacting with the clock.
