User

Creates the User class that handles all functionality for an app user across all screens and launch instances of the app.

Declaration

class User

Parameters

Handles tracking the location of a user’s device and retrieve their current location.

shared

Instance of the User object for app to access data within the class.

buoyBluetoothManager

Manages all Bluetooth functionality for a Buoy.

spoolerBluetoothManager

Mannages all Bluetooth functionality for a Spooler.

alertText

Text to be shown on an alert if the user does an undoable action or there is an issue with a feature at the time of action

shouldShowAlert

Boolean variable to let the app knowif it should show the alert. If true, app displays the alert.

location

Current location of the user.

connectedBuoys

Array of Buoys that the user is currently connected to.

disconnectedBuoys

Array of Buoys that the user is currently disconnected from but have connected to previously.

selectedBuoy

Stores which Buoy the user currently has selected.

selectedString

Stores which String the user currently has selected.

buoyStrings

Array of all of a user’s Buoy Strings.

prevConnectedBuoys

Dictionary of all Buoys that a user has previously connected to. It stores the original name that is given by Bluetooth as the key and value dy default. When a user changes the name of the Buoy within the app, it changes the value for that Buoy to store the friendly name of the Buoy.

Instance Methods

func deployBuoy(buoy: Buoy, retrievalTime: Date, buoyString: BuoyString)
When a user presses the button to deploy their Buoy after setting a retrieval time for it, this function is called.
func disconnectBuoy(buoy: Buoy)
When a user clicks on the button to disconnect from a Buoy on the spec screen, this function is called. This function removes it from the list of connected buoys, adds it to the list of disconnected buoys, and terminates the Bluetooth connection with the device
func encode(to: Encoder) throws
Encodes the different arrays that will need to be persistent across app launches for a user.
func forgetBuoy(buoy: Buoy)
When a user clicks on the button to forget a Buoy on the spec screen, this function is called. This function iterates over all previously connected buoys. It removes the buoy from the list of previously connected buoys, its respective list if it was connected or disconnected, and, if it is currently connected to, terminates the Bluetooth connection.
