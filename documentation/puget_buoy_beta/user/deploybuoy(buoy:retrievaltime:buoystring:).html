Instance Method

deployBuoy(buoy:retrievalTime:buoyString:)

When a user presses the button to deploy their Buoy after setting a retrieval time for it, this function is called.

Declaration

func deployBuoy(buoy: Buoy, retrievalTime: Date, buoyString: BuoyString)

Parameters

buoy

Takes in a Buoy that the user has selected and wants to deploy.

retrievalTime

Time the user wants to have the Buoy resurface.

buoyString

Buoy String the user wants to have the Buoy attached to.
